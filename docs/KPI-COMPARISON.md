# Atlantico ConstruData - KPI: Antes vs Depois

> Documento de medição de impacto operacional por módulo.
> Compara o fluxo **manual** (sem plataforma) com o fluxo **digitalizado** (com ConstruData).

---

## Metodologia

Para cada módulo, medimos:
- **Tempo gasto** (min/tarefa)
- **Frequência** (vezes/mês)
- **Custo operacional** estimado (horas-homem * custo/hora do profissional)
- **Taxa de erro/retrabalho** (%)
- **Economia mensal projetada** para uma obra tipo Sabesp (CT 11481051 - 40+ funcionarios)

|--------|---------|
| Engenheiro de campo | R$ 85 |
