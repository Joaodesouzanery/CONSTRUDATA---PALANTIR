# Atlântico ConstruData — Scripts para Vídeos Remotion

> Cada módulo abaixo tem um script/prompt pronto para usar em uma sessão Claude + Remotion.
> Copy-paste o prompt do módulo desejado para gerar o vídeo.

---

## Especificações Técnicas (para todos os vídeos)

```
- Resolução: 1920x1080 (16:9)
- FPS: 30
- Duração: 60-90 segundos por módulo
- Fundo: #1f1f1f (dark mode)
- Accent: #f97316 (orange)
- Texto principal: #f5f5f5
- Texto secundário: #a3a3a3
- Bordas: #525252
- Cards: #2c2c2c ou #3d3d3d
- Font: Inter ou system-ui (corpo), Space Grotesk (títulos)
- Animações: fade-in suave (300ms), slide da esquerda para passos sequenciais
- Logo: gota d'água com contorno laranja (SVG inline)
```

## Notas para Produção

### Ordem sugerida de produção:
1. **LPS/Lean** — mais diferenciador, maior impacto de venda
2. **Torre de Controle** — apelo visual forte (dashboard)
3. **Suprimentos** — dor clara (dinheiro perdido em divergências)
4. **RDO** — fácil de entender (papel → digital)
5. **Planejamento** — Gantt é visual e familiar
6. **Medição** — específico para saneamento/Sabesp
7. **Relatório 360** — complementar

### Elementos reutilizáveis entre vídeos:
- Abertura com logo (5s) — mesmo template
- CTA final (10-15s) — mesmo template
- Fundo dark mode (#1f1f1f) — consistente
- Accent orange (#f97316) — consistente
- Animação de "problema → solução" — padrão reutilizável
